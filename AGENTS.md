# AGENTS.md

## Repo Goal

This repository is a portfolio-grade project intended for recruiter review and interview discussion.

## Working Rules

- Start with a plan before major code changes.
- Keep changes small and reviewable.
- Run lint, tests, and type checks before marking milestones complete.
- Keep README and docs in sync with the code.
- Prefer stable, well-supported libraries.
- Do not invent metrics or business impact.

- Ruff passes
- Black passes
- Mypy passes for core modules
- Pytest passes where tests exist
- Imports are clean
- README reflects actual current behavior
